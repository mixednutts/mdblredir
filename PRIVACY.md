# Privacy Policy

**MDBList Redirector** does not collect, transmit, or share any personal data.

- Your MDBList API key is stored locally in `chrome.storage.local` and is never sent anywhere except directly to the MDBList API (`api.mdblist.com`) via a `?apikey=` query parameter.
- No analytics, tracking, or telemetry are used.
- No data is synced to external servers or cloud services.

If you have questions, open an issue on the GitHub repository.
